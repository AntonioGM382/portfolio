Project Overview

This application provides a comprehensive platform for pet owners, enabling them to manage pet profiles, review services offered by other users, and access pet care services such as dog walking or overnight stays.

Additional features include a pet food search, user profile management, and easy service acquisition tailored to their pets' needs.

New Functionalities

  • Search and hire services for dog walking.
  • Review other users to foster trust within the community.
  • Locate overnight care services for pets.

Technologies Used

  • React Native for cross-platform mobile development.
  • Firebase for backend services and real-time database integration.
  • Cloudinary for secure image storage and optimization.
  • Custom authentication for enhanced user security.

Features

  • Interactive pet profiles for seamless management.
  • Tailored food recommendations with purchasing options.
  • Integrated user-to-user messaging system.
  • Comprehensive service search for pet care needs like dog walking and boarding.
