Project Overview

This application offers a comprehensive platform for pet owners, including managing pet profiles, reviewing other users, and accessing pet services such as dog walking or overnight stays at others' homes.

It also allows users to search pet food, manage user profiles, and view personalized ads and messages.

New Functionalities

  • Search and hire services for walking dogs.
  • Review users to build trust within the community.
  • Search for overnight care services for pets.

Technologies Used

  • React Native for mobile development
  • Firebase for backend and real-time database
  • Cloudinary for image storage
  • Custom user authentication

Features

  • Interactive pet profiles
  • Food recommendations and purchasing
  • User-to-user messaging system
  • Ads tailored to user preferences
  • Searchable services for dog walking and pet boarding
