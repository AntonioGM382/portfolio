Project Overview

Against All is a full-stack multiplayer game leveraging modular programming with ReactJS, Node.js, and Kafka. It includes real-time gameplay, interactive UI, and APIs for login, registration, and user updates.

The project also implements data encryption using AES and API integration with OpenWeather for dynamic game elements.

Technologies Used

  • ReactJS (with mongoose, bcryptjs, socket.io)
  • AWS Lightsail for deployment
  • OpenWeather API
  • MongoDB
  • Kafka for message handling

Features

  • Dynamic board generation with weather data for zones
  • Real-time logs and user updates
  • Encryption for secure data transmission
  • Dockerized deployment on AWS Lightsail
