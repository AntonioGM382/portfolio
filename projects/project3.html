Project Overview

The ToDoList project is a feature-rich task management system built using agile methodologies. It includes advanced features such as tasks and subtasks, tagging, filtering, and more, tailored for efficient team collaboration and personal task management.

Technologies Used

  • Java Spring Boot
  • Thymeleaf
  • Docker
  • PostgreSQL
  • Spring Security

Features

  • Drag and drop tasks to update states (ToDo, In Progress, Done).
  • Tagging and filtering tasks.
  • Weekly calendar view for tasks.
  • Password recovery and user account management.
  • Task assignment to teams.

Additional Features

The system enables real-time collaboration, progress tracking, and integration with external tools for seamless productivity management.
