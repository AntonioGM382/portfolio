Capacity Planning Tool

Workforce scheduling platform built for a leading automotive enterprise to replace legacy systems.

System Architecture

High-performance static SPA (Next.js 16, React 19, Tailwind v4, TanStack Query) backed by a robust Django 6 & MySQL monolith deployed via AWS ECS Fargate.

Dual-Layer Engine

Generates 6 months of live schedules dynamically from master shift blueprints, safely handling daily disruptions (sick leave, branch swaps) without erasing recurring patterns.

Enterprise Features

Real-time multi-country RBAC permissions, complex biweekly even/odd week parity logic, and automated vacation approval workflows that auto-prevent shift conflicts.
